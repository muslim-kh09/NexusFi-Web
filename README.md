# NexusFi Web

🌍 **Live Demo:** [nexusfi-web.vercel.app](https://nexusfi-web.vercel.app)

The official landing page and web presentation for **NexusFi** — a self-contained, 3-tier Micro-ISP and Local Cloud Ecosystem engineered to run natively on rooted Android devices.

## 🌟 Vision
This landing page is built using a "Kinetic Minimalism" approach. It features an ultra-premium, dark "obsidian" aesthetic, powered by dynamic glassmorphism and ambient blurred lighting.

A central element of the design is the interactive feather, moving fluidly with the cursor to symbolize the "weightlessness" of the NexusFi infrastructure, shifting elegantly against the dense architectural details of the system.

## 🚀 Technologies Used
- **React** (Vite)
- **Tailwind CSS v4** (Advanced glassmorphism & typography scaling)
- **TypeScript**
- **Phosphor Icons**

## 🌍 Bilingual Support (i18n)
The application supports seamless transitions between English (LTR) and Arabic (RTL).
- Custom typography adjustments ensure that Arabic headers scale proportionally to maintain their majestic aesthetic.
- Layouts and animations dynamically adapt based on reading direction (`dir="rtl"`).

## ⚙️ Features Highlighted
- **PENTA-SHIELD**: 5-signal device identity system surviving MAC randomization.
- **Ghost Radar**: Background daemon for L2 physical deauthentication.
- **Smart Storage**: USB OTG auto-discovery and WebSocket hardware telemetry.
- **DNS Sovereignty**: DoH/DoT suppression and HTB bandwidth shaping.
- **Local Cloud**: Offline FastAPI media server with zero internet bandwidth usage.

## 🛠️ Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
